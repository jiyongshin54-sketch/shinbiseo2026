### 직원 관리 (blanchedalmond 헤더 배경)

MembersGV (max-height:254px):
| 직위 | 직원 이름 | 핸드폰 번호 | 상태 | 입사 신청일 | 입사 승인자 | 입사 승인일 | [승인] | [삭제] |

- **직위**: 대표 / 관리자 / 직원
- **상태**: 승인 / 신청 / 삭제
- **[승인]** 버튼: 상태가 "신청"인 직원을 "승인"으로 변경 (대표만)
- **[삭제]** 버튼: 직원 삭제 (대표만, users_deleted 테이블로 이동)
- EmptyDataText: "직원관리는 대표만 볼 수 있습니다."

## 별도 MyCompany.aspx (판매회사 전용 관리 페이지)

별도 페이지로 접근. 자체 헤더(cadetblue) + 메뉴 5개:

### 메뉴 (cornflowerblue 배경)
```
상품 관리 | 재고 관리 | 거래처 관리 | 직원 관리 | 신BS로 돌아가기
```

### 상품관리PH
ProductsGV (max-height:400px):
| 품명 | 색깔 | 두께 | 사이즈 | 마대량 | 3.8급 | 3.9급 | ... | 5.0급 | [관리] |

- 13단계 등급별 단가 표시 (UnitPrice01~UnitPrice13)
- `[관리]` 클릭 → 하단 추가/수정 폼 표시

상품 추가/수정 폼 (AddProductTable, palegoldenrod 배경):
```
행1: 상품ID, 품명, 색깔, 두께, 사이즈, 마대량
행2: 3.8~4.3 단가 (각 60px)
행3: 4.4~5.0 단가
버튼: [취소] [상품 추가]
```

### 거래처관리PH
CustomersGV (max-height:370px):
| 상호 | 사업자번호 | 대표 | 전화번호 | 팩스번호 | 이메일 | 담당자 | 업태 | 종목 | 일반(등급) | 마대(등급) | 온라인(CompanyID) | [관리] |

거래처 추가/수정 폼 (AddCustomerTable, palegoldenrod 배경):
```
행1: 거래처ID, 사업자번호, 상호*, 대표*, 담당자*
행2: 전화번호, 팩스번호, 이메일, 주소
행3: 일반등급*[3.8~5.0급▼], 마대등급*[3.8~5.0급▼], 업태, 종목, 홈페이지
행4: 비고, 온라인
```

### 직원관리PH
MembersGV: 직원이름, 핸드폰번호, 직위, 상태, 입사신청일, 입사승인자, 입사승인일, [승인], [삭제]

## API

### PATCH /api/companies/[companyId]
- 회사 정보 수정

### GET /api/users
- params: company_id
- 직원 목록 조회

### PATCH /api/users/[userId]/status
- 직원 승인/거절/삭제

### GET /api/products
- params: seller_id
- 상품 목록 조회

### POST /api/products
- 상품 추가

### PATCH /api/products/[sellerId]/[productId]
- 상품 수정

### DELETE /api/products/[sellerId]/[productId]
- 상품 삭제
