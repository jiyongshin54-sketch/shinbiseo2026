# Step 5-1: 맞춤품 주문 (스크린샷 기준 정확 레이아웃)

## 개요
기성품이 아닌 맞춤 제작 상품을 주문하는 화면.
두께/폭/길이/인쇄 등 스펙을 직접 입력하고 단가가 자동 계산됨.

---

## 레이아웃 (스크린샷 기준)

### 상단: 거래 대상 선택 바
```
맞춤품 주문
구분: [판매▼]         거래처: [________] [Guest(대표)    ▼]
```

### 중단: 좌우 2분할

**좌측 (~25%)**: 맞춤품 입력 폼 (background: whitesmoke)
```
┌─────────────────────────────────┐
│ *카테고리: [카테고리를 선택하세요..▼] │
│ 품명:  [________]  색깔: [____]  │
│ *두께: [________]                │
│ *사이즈: [____] * [____]        │  (폭 × 길이)
│ 도수:  [무지▼]  인쇄명: [______]  │
│ 가공방식: [____________________] │
│ 도안:  [첨부]                    │  (FileUpload)
│ 예상단가: [______]               │  (LinkButton 클릭으로 계산)
│ 수량:  [________]               │
│ 예상금액:              0         │
│ [초기화]              [추가]     │
└─────────────────────────────────┘
```

- **카테고리 DDL**: Categories 테이블에서 로드
- **도수 DDL**: [무지, 1도, 2도, 3도, 양면1도, 양면2도, 양면3도]
- **예상단가**: LinkButton (클릭하면 가격 공식으로 단가 계산)
  - 계산 후 TextBox에 표시 (수정 가능)
- **도안**: FileUpload (파일 첨부 → 서버 업로드)
- `[추가]` 클릭 → 우측 장바구니에 추가
- `[초기화]` 클릭 → 폼 초기화

**우측 (~75%)**: 장바구니 + 금액/상태 (background: skyblue)

장바구니 GridView (맞춤CartGV):
| 순번 | Category | 품명 | 색깔 | 두께 | 사이즈 | 인쇄명 | 도수 | 가공방식 | 도안 | 단가 | 수량 | 금액 | [수정] | [빼기] |

금액/상태 영역:
```
┌─────────────────────────────────────────────────────────────┐
│ 거래일자: [2026.03.19]               주문번호: [________]    │
│ 금액합계:              0  금액조정: [+▼] [0______]          │
│ 최종공급가:            0  부가세:   [▼]  [0______]          │
│ 총계:                 0  상태:     [________]              │
│ 비고: [__________________________________________________]  │  (MultiLine, Rows=5)
│                                                             │
│    [견적 등록]           [주문 등록]           [취소]          │
└─────────────────────────────────────────────────────────────┘
```

- 비고: MultiLine TextBox (Rows=5, 기성품보다 큼)
- **3개 버튼**:
  - `[견적 등록]`: 상태="견적"으로 등록 (구매회사가 견적 요청 시)
  - `[주문 등록]`: 상태="주문"으로 등록 (판매회사가 직접 등록 시)
  - `[취소]`: 장바구니 초기화

### 하단: 안내문 (orange 배경)
```
┌─ 맞춤품 주문 방법 안내 ─────────────────────────────────────────────┐
│ * 맞춤품은 이 화면에서 견적요청 한 후 판매회사의 견적응답을 보고       │
│   메인 화면에서 꼭 주문을 하셔야 합니다. *                           │
│ 1. 견적 요청은 카테고리, 두께와 사이즈 필수값만 입력 후 상품을        │
│    추가하여 견적 요청하시면 됩니다.                                  │
│ 2. 특정 건이 아니면 입력 사항을 다 입력 하신 후 예상 단가를 클릭하면  │
│    단가가 계산됩니다.                                               │
│ 3. 견적 요청 하신 후에 메인 화면에서 견적 응답을 확인 하시고 꼭       │
│    주문 처리 해 주시기 바랍니다.                                     │
└────────────────────────────────────────────────────────────────────┘
```

## 맞춤품 가격 공식

### 기본 공식
```
단가 = ValueA + ValueB

ValueA (재료비) = 두께 × 폭 × (길이 + 5.0) × 0.184 × 등급상수
ValueB (인쇄비) = max((도수값 × 8000 × 폭) / 45700, 2.0)
  (도수가 "무지"이면 ValueB = 0)
```

### 등급상수 (gradeToNumber)
| 등급 | 상수 |
|------|------|
| 3.8급 | 3.8 |
| 3.9급 | 3.9 |
| ... | ... |
| 5.0급 | 5.0 |

### 도수 매핑
| 도수 | 도수값 |
|------|--------|
| 무지 | 0 |
| 1도 | 1 |
| 2도 | 2 |
| 3도 | 3 |
| 양면1도 | 2 |
| 양면2도 | 4 |
| 양면3도 | 6 |

### 예상단가 링크 클릭 동작
1. 두께, 폭, 길이, 도수 값 읽기
2. 고객 Level3 등급 → 등급상수 변환
3. ValueA + ValueB 계산
4. 소수점 2자리 반올림
5. 예상단가 TextBox에 표시
6. 수량 × 단가 = 예상금액 표시

## 견적 프로세스 (맞춤품 전용)

```
[구매회사]                    [판매회사]
맞춤품 화면에서 견적 등록  →  전광판에 "견적" 상태로 표시
                            ← 견적 응답 (단가 확정)
메인 전광판에서 "주문" 확정 →  "주문" 상태로 변경
```

- 구매회사: `[견적 등록]`만 사용 가능 (주문 등록 불가)
- 판매회사: `[견적 등록]`, `[주문 등록]` 모두 사용 가능
- 견적 응답은 메인 전광판의 주문 상세에서 처리
