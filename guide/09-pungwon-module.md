# Step 8: 풍원 딜러네트워크 (스크린샷 기준 정확 레이아웃)

## 개요
Pungwon.aspx 클론. 풍원비닐포장(CompanyID='00002') 전용 딜러네트워크.
Main.aspx와 **별도 페이지**로 독립 운영됨.
풍원 직원 + 풍원과 계약된 구매회사(딜러)가 접근 가능.

## 화면별 레이아웃

### 1. 기성품 주문 (기성PH) - Main의 기성품 주문과 유사

**좌측 (48%)**: background:whitesmoke
```
[품명, 색깔, 사이즈로 조회 가능_________] [조회]
```
주문후보GV (max-height:612px):
| 품명 | 색깔 | 두께 | 사이즈 | 마대수량 | 일반가 | 마대가 | 수량(TextBox+"00") |

- 일반가/마대가: TemplateField (해당 고객 등급에 따른 단가)

**우측**: background:skyblue
기성CartGV:
| 품명 | 색깔 | 두께 | 사이즈 | 단가 | 수량 | 금액 | 묶음DDL[공란,1-5] | [빼기] |

합계:
```
공급가: xxx (16%)  부가세: [DDL▼] (16%)  총계: xxx (16%)
비고: [________________] (MultiLine, Rows=5)
[주문] (BackColor:CadetBlue)
```

**안내문** (orange 배경): 기성품 주문 방법

※ Main.aspx 기성품 주문과 차이점:
- 거래일자/주문번호/금액조정/상태DDL 없음 (단순화)
- 구매회사가 직접 사용하는 화면이므로 간소화

---

### 2. 맞춤품 주문 (맞춤PH)

**좌측 (25%)**: 입력 폼
```
품명: [________]  색깔: [______]
*두께: [________]
*사이즈: [____] * [____]  (폭×길이)
도수: [없음▼] (없음,1도,2도,3도,양면1도,양면2도,양면3도)
납기일: [________]
인쇄명: [________]
도안: [FileUpload]
가공방식: [________]
예상단가: [________]
수량: [________]
예상금액: 0
[추가]
```

※ Main.aspx 맞춤품과 차이:
- **납기일** 필드 추가 (Main에는 없음)
- **카테고리 DDL 없음** (Main에는 있음)

**우측 (75%)**: background:skyblue
맞춤CartGV:
| 품명 | 색깔 | 두께 | 사이즈 | 인쇄명 | 도수 | 가공방식 | 납기일 | 단가 | 수량 | 금액 | [빼기] |

합계:
```
예상공급가: xxx  예상부가세: [DDL▼]  예상총계: xxx
[견적 요청 하기] (BackColor:CadetBlue)
```

※ 구매회사 전용이므로 "견적 요청"만 가능 (주문 등록 불가)

---

### 3. 기성품 단가표 (스크린샷 확인됨!)

**구매회사(딜러)용**: 상품PH
ProductsGV (max-height:1200px):
| ID | 카테고리 | 품명 | 색깔 | 두께 | 사이즈 | 마대량 | 일반단가 | 마대단가 |

- 해당 고객의 Level1/Level2 등급에 맞는 단가만 표시

**풍원 관리자용**: 풍원상품PH (스크린샷 = 이 화면!)

제목: "풍원 기성 상품 관리"

PungwonProductsGV (max-height:1200px):
| ID | 카테고리 | 품명 | 색깔 | 두께 | 사이즈 | 마대량 | 3.8급 | 3.9급 | 4.0급 | 4.1급 | 4.2급 | 4.3급 | 4.4급 | 4.5급 | 4.6급 | 4.7급 | 4.8급 | 4.9급 | 5.0급 | [관리] |

- **13개 등급별 단가 컬럼** (UnitPrice01~UnitPrice13)

### 4. 고객가입 (고객가입PH)

온라인 고객 가입 신청 폼 (palegoldenrod 배경):
```
행1: 풍원고객ID(비활성), 사업자번호, 상호*, 대표*, 담당자
행2: 전화번호, 팩스번호, 이메일, 주소
행3: 업태, 종목, 홈페이지, 비고
버튼: [취소]  [온라인 거래 신청]
```

- 구매회사가 풍원과 온라인 거래를 신청하는 폼
- 승인되면 Customers + Contracts 테이블에 등록

## API

### GET /api/products?seller_id=00002
- 풍원 상품 목록 (등급별 전체 단가 or 고객 등급별 단가)

### POST /api/products
- 상품 추가

### PATCH /api/products/[sellerId]/[productId]
- 상품 수정 (단가 변경 포함)

### DELETE /api/products/[sellerId]/[productId]
- 상품 삭제

### POST /api/customers (고객가입 신청)
- 풍원 고객으로 등록 신청
